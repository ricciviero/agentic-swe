# Resources

Use current primary sources when planning real experiments; LLM tooling changes quickly.

## Study Path

1. Karpathy nanoGPT: https://github.com/karpathy/nanoGPT
2. Sebastian Raschka, "Build a Large Language Model (From Scratch)": https://www.manning.com/books/build-a-large-language-model-from-scratch and https://github.com/rasbt/LLMs-from-scratch
3. Lightning AI LitGPT: https://github.com/Lightning-AI/litgpt
4. Hugging Face Transformers and tokenizers documentation.

## Small-Model References

- SmolLM2: https://huggingface.co/HuggingFaceTB/SmolLM2-1.7B
- SmolLM2 paper: https://arxiv.org/abs/2502.02737
- TinyLlama, Pythia, and GPT-2 as historical baselines.
- FineWeb-Edu dataset: https://huggingface.co/datasets/HuggingFaceFW/fineweb-edu

## Local and Apple Silicon

- MLX: https://github.com/ml-explore/mlx
- MLX examples: https://github.com/ml-explore/mlx-examples

## Data Notes

For Italian SFT/RLHF, curate or adapt licensed instruction-response and preference data. Do not import private customer data or unlicensed corpora.

URLs can remain stable while repository APIs change; verify project status before using a source in a new experiment.
